# Testing with Jest Demo

This is a starter repository for learning unit testing with Jest.

## Features
- Utility functions
- Validation functions
- Authentication-related logic
- Cart and discount calculation logic

## Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd testing-with-jest-demo
```


## About this repository

This repository is a starter project for learning **unit testing with Jest**. It contains a small Node.js codebase with utility functions, validation logic, authentication-related functions, and cart/discount business logic.

The project is intentionally kept simple so that you can first understand the codebase, run it locally, and then gradually learn how to write tests for different kinds of functions and services.

This repository is designed for classroom use where the testing setup is done step by step during the lecture. Jest is not pre-configured so that you can learn how to install it, configure it, create test files, and run tests on their own.

By working on this repo, you will learn how to test:
- simple utility functions
- input validation logic
- authentication logic
- cart and pricing related business logic
- positive, negative, and edge cases
